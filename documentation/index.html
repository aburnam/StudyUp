Install

1. Import SQL

- Dump SQL file database\openquiz.sql to your database.

- Open app\Config\database.php and config follow your database connection.

2. Apache and mod_rewrite

- Please read this article

3. Set folder writable

- Please make sure folder app/tmp writable

4. Social Config

- Facebook: Go to https://developers.facebook.com/apps create your app (Remember: Add your domain and check allow login option). Then open app\Config\facebook.php and update follow your facebook app config.

- Twitter: Get an api_key and secret from twitter (https://dev.twitter.com/docs/auth) and update file app/Config/twitter.php.

5. Google analytics

- Please read how to get google analytics profile number here and update file app/Config/google_analytics.php.
